## Vocaloid 曲
在 Maimai 中，Vocaloid 曲目占了大半江山。从初次打 Maimai 的新人到虹框大佬，从 1 级谱面到 14 级谱面，都能看见 Vocaloid 曲目的身影。

Vocaloid 是一款合成歌声的软件，只需要输入音高和歌词，就可以合成原为人类声音的歌声。当然，比起这个名字，大多数人可能更熟悉的是“初音未来”和《甩葱歌》，或是 ilem 的《普通 Disco》《达拉崩吧》、“言和”、“洛天依”、Bilibili 拜年祭的《九九八十一》《权御天下》等。相关的介绍文章很多，此处就不再展开了。

今天的 Vocaloid 曲目的风格百花齐放。由みきとP创作的《**ロキ**》（Roki，[BV1AW411t7Dx](https://www.bilibili.com/video/BV1AW411t7Dx)）正如曲名般有着摇滚少年和少女一样酷帅的感觉。

{% mmedia "bilibili" "bvid:BV1AW411t7Dx" %}

而 wowaka 创作的《**裏表ラバーズ**》（里表恋人，[BV1xx411c7RL](https://www.bilibili.com/video/BV1xx411c7RL)）则以高速的歌词（所谓“高速神言”）著称。高速的歌词也是相当多 Vocaloid 歌曲的特征。

{% mmedia "bilibili" "bvid:BV1xx411c7RL" %}

当然，也有曲速较慢的感伤风格的 Vocaloid 曲，例如 DECO\*27 创作的《**妄想感傷代償連盟**》（妄想感伤代偿联盟，[BV1hs411W7uR](https://www.bilibili.com/video/BV1hs411W7uR)）。

{% mmedia "bilibili" "bvid:BV1hs411W7uR" %}

うどんゲルゲ的《**どうしてこうなった**》（为什么会变成这样呢，[BV1ks411k7ja](https://www.bilibili.com/video/BV1ks411k7ja)）则是一首旋律魔性、歌词毫无意义的“电波系”歌曲。

{% mmedia "bilibili" "bvid:BV1ks411k7ja" %}

除此之外，Maimai 还收录了很多很多的 Vocaloid 歌曲，相当一部分有名的 Vocaloid 歌曲都能在 Maimai 中玩到。著名 Vocaloid 创作者ナユタン星人更是为 MaimaiDX 编写了原创曲《超常マイマイン》（超常 Maimain，[BV154411A7p2](https://www.bilibili.com/video/BV154411A7p2)）。

{% mmedia "bilibili" "bvid:BV154411A7p2" %}

如果你是一名 Vocaloid 爱好者，或许也可以在 Maimai 中收获很多打歌的乐趣。

## 音游曲
作为音游，收录曲目自然还是音游曲居多。音游曲的风格是怎么样的呢？和电子舞曲一样吗？

比如说 Nankumo 的《**Dragonlady**》（[BV1ks411R7hq](https://www.bilibili.com/video/BV1ks411R7hq)），可以算是比较“通常向”的电子舞曲了。这首歌因为其名字和早期某网络事件中的主角雷同而“风评被害”，再加上其魔性的人声和 BGA，成为了各大二次创作的常客。但不可否认的是，本曲和 BGA 确实都很优秀，配得上音游大会 BOF2009 的冠军的名号。

{% mmedia "bilibili" "bvid:BV1ks411R7hq" %}

不过比起一般的电子舞曲来说，音游曲通常会更激情一些。KAH 的《**energy trixxx**》（[BV1Tt411d7bF](https://www.bilibili.com/video/BV1Tt411d7bF)）的风格是音游曲中最常见的风格之一——Hardcore（硬核）。这样的歌曲富有激情、节奏性强，比大众所知的电音（如《Faded》等）也要更快一些，更硬核一些。

{% note warning %}
**WARNING**：该视频开头含有高频闪光，光敏癫痫患者等请谨慎查看。
{% endnote %}

{% mmedia "bilibili" "bvid:BV1Tt411d7bF" %}

如果上面的还不够硬核，那么可以听听 USAO 的《**Blows Up Everything**》（[BV1Nt411K7FX](https://www.bilibili.com/video/BV1Nt411K7FX)）。USAO 作为知名同人音乐社团 HARDCORE TANO*C 的成员，擅长制作 Frenchcore（一种曲风）类型的歌曲，在人声切片的运用和 Bass 的设计方面有着高超的技术。

{% mmedia "bilibili" "bvid:BV1Nt411K7FX" %}

速度不够快？试试 BPM 高达 257 的《**QZKago Requiem**》（[BV1JQ4y1T7qo](https://www.bilibili.com/video/BV1JQ4y1T7qo)）。这首歌曲出自同属于社团 HARDCORE TANO*C 的 t+pazolite，后者给 Maimai 提供了《**Garakuta Doll Play**》、《**Oshama Scramble!**》和《**QZKago Requiem**》等在玩家中热度极高的曲目。

该曲的曲风为 Speedcore。通常来讲，这也是魔王曲的常见曲风——极高的 BPM 和激烈的鼓点，可以更容易创作出高难度的谱面。

{% mmedia "bilibili" "bvid:BV1JQ4y1T7qo" %}

音游曲中也有不那么激烈的歌曲。削除创作的 **Altale**（[BV1sx411P7Le](https://www.bilibili.com/video/BV1sx411P7Le)）并不激烈，BPM 甚至不超过 90，却依旧凭借史诗感和出色的编曲一举夺得音游大会 G2R2014 的冠军。

{% mmedia "bilibili" "bvid:BV1sx411P7Le" %}

sasakure.UK 创作的 **Jack-the-Ripper◆**（[BV1Js411Z7Y6](https://www.bilibili.com/video/BV1Js411Z7Y6)）则运用了反复变化的 BPM 和拍数、不安的旋律和嘈杂的电吉他表现了开膛手杰克的疯狂和“优雅”。
